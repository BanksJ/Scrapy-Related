# Scrapy小练习

### 项目简述
    增量抓取车300、淘车平台二手车发布数据

### 技术点
* 多爬虫并行增量抓取
* 配置代理池，突破IP限制
* cookie池自动化更新
* 个性化配置爬虫参数
* 策略化分发目标网址
* XPath提取数据，管道清洗
* 可拓展加入自启动，实现7*24抓取
* 日志记录
* 写入MySQL

### 优化方向
* 效率：优化爬虫抓取策略以及代理池分发方式，提升抓取速度
* 资源：去重策略进一步优化（Bloomfilter），降低资源消耗
* 技术：采取redis去重，搭建分布式爬虫，完成数据抓取
